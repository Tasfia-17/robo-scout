# Web Agents Hackathon Submission

## Project Info

**Name:** RoboScout  
**Description:** AI web agent that autonomously manages robotics fleets by interacting with enterprise dashboards and operational tools in real-time

**Tech Stack:** TinyFish API, Gemini 3 Flash, MuJoCo WASM, Node.js

## Links

- **Demo:** https://robo-scout.vercel.app/
- **GitHub:** https://github.com/Tasfia-17/robo-scout
- **Submit:** https://forms.gle/VdDDP1fADVLiWE5MA

## Key Features

- TinyFish web automation (scrape dashboards, weather, maintenance)
- Gemini AI decision-making every 2 seconds
- 3D digital twin visualization
- Two-way automation (read + write to websites)
- Multi-robot fleet coordination
